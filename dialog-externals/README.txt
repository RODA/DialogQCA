Dialog externals in this directory are owned by DialogQCA.

Use this directory for DialogQCA-specific dialog external-call logic that does not
belong in shared provider code.

If a DialogQCA external is reusable by more than one product and it is R-specific,
move it under shared/runtime/providers/r/dialog-externals/.

If it is reusable by all products and all providers, move it under
shared/dialog-externals/.
